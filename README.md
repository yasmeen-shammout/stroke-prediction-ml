#stroke Prediction Using Machine Learning
This project builds a model to predict stroke risk based on patient health data.

# Tools
Python, Pandas, Scikit-Learn, Power BI
# Steps
- Data cleaning and preprocessing
- Feature engineering
- Model training (Gradient Boosting & Random Forest)
- Achieved 89.6 % accuracy

# Key Insight
Age, BMI, and glucose level were top predictors of stroke risk.
